# Jazztips

[Visit site](https://jazztips.se/)

### Features

+ Jekyll 3
+ Fast
+ Tags
+ RSS Feed
+ Hosted on Amazon S3 (Frankfurt datacenter)
+ using s3 cli 
+ Typeface Nitti
+ Using blazy for lazy loading images
+ Images hosted on Cloudinary CDN
+ Musicans using nested YAML
+ HTTPS everywhere

## Setup

    bundle install

## Run

    npm start

    npm serve

## Deploy to AWS S3/Cloudflare

    npm run deploy
