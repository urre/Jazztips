# [Jazztips](https://jazztips.se/)

### Features

+ Jekyll 3
+ Blazing fast search
+ Tags
+ Using blazy for lazy loading images
+ Images hosted on Cloudinary CDN
+ RSS Feed
+ Hosted on Netlify

## Setup

    bundle install

## Run

    npm start

    npm serve

## Get music data for a new post

    npm run new

> You'll need a `.env` file with Cloudinary API keys, check `.env-example`
